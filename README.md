# 概要
GodotエディタからNeovimを一発で起動する為のシンプルなプラグイン。<br>Windows専用。

# インストール
godotプロジェクトディレクトリのaddonsに置く

# 使い方
<img src="./images/open_nvim_button.png" alt="usage image"/>
右上のボタンを押す
